数组与类数组的区别

类数组定义

  1. 拥有 length ,其它属性（索引）为非负整数（对象中的索引会被当做字符串来处理）
  2. 不具有数组所具有的方法;
  3. 类数组是一个普通对象，而真实的数组是 Array 类型。

常见的类数组有:

  1. 函数的参数 arugments ;
  2. DOM 对象列表(比如通过 document.querySelectorAll 得到的列表);
  3. jQuery 对象 (比如 $("div"));

类数组可以转换为数组

Array.from方法用于将两类对象转为真正的数组：类似数组的对象(array-like object)和可遍历 (iterable)的对象。
