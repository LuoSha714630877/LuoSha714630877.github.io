Typescript学习入门(一)变量与数据类型
发表于 2020-07-23| 更新于 2020-07-23|Typescript
字数总计:397|阅读时长: 1 分钟
|阅读量:

今天正式开始学习Ts，之前断断续续学习过其语法知识，但是没有总结以及做笔记，那么今天开始认真做课代表吧!

变量

跟js不同的是，我们在变量声明时需要指明数据类型:
let dName:string = 'Mondy';
当指定了类型后，就只能给这个变量设置相同类型的值，否则会报错:
dName = 520 //报错

数据类型

在ts语法中一方面保留了js原有的数据类型：

  • string let aName:string = '吕布'
  • number let dAge:number = 18
  • boolean let isBoy:boolean = true 注意：在ts中不能用1、0代替布尔值
  • Array 需要在声明时指定数组中元素的类型
    方式一：let 数组名:类型[] = [值1,值2] 即 let arrHeros: string[] = ['盖伦','卡特','艾希']
    方式二：let 数组名:Array<类型> = [值1,值2] 即 let arrHeros:Array<string> = ['盖伦','卡特','艾希']
  • null let nul:null = null
  • undefined let undef:undefined = undefined
  • Symbol es6中新增类型,本质上是一种唯一标识符
    let id = Symbol("id“);
    其有两个特点,一个是唯一性,即使是用同一个变量生成的值也不相等 另一个是隐藏性,for···in，object.keys() 不能访问
  • Object

同时，新增了几种类型:

  • tuple 元组
  • enum 枚举
  • any 任意类型
  • never
  • void
